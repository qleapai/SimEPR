# Reference Spectra Plan

Future DOI-linked SimEPR releases should include curated experimental reference spectra and parallel SimEPR/EasySpin fits for:

- TEMPO or a TEMPO-like nitroxide reference;
- DPPH g-calibration reference;
- Mn(II) sextet reference;
- Cu(II) axial powder reference;
- vanadyl/V(IV) axial powder reference;
- representative PBN adducts including PBN-OH, PBN-CH3/DMSO, PBN-OOH/O2-derived adducts, and selected isotope-labelled N2RR/CO2RR candidate controls where available.

Each reference entry should include:

- raw spectrum;
- processed spectrum;
- spectrometer metadata;
- literature or experimental source;
- SimEPR fit;
- EasySpin fit;
- exported metrics and uncertainty table;
- known limitations and expected valid use.

Until those reference spectra are curated, built-in SimEPR components should be treated as screening models.
